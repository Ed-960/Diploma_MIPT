Граф меню (RAG, демо-вид) Загрузка…

В демо-режиме на холсте при обзоре без подписей — наведите на узел, название во всплывающей подсказке. При приближении появляются подписи и больше слабых рёбер; при отдалении — снова чище. Клик по графу, колёсико или + / −; панорама — левая кнопка на пустом фоне.
