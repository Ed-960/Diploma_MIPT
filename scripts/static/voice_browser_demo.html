Drive-through (browser speech)

Push-to-talk (hold the mic button) or hands-free (mic always on; your line sends after ~1.4s of silence — not true duplex, but no button per phrase). Best with headphones in hands-free so speakers do not feed back into the mic. Uses Web Speech API (Chrome / Edge). URL: http://127.0.0.1:8765/. runtime.lastError in the console is often a browser extension, not this app.
Mic language must match how you speak: Russian → choose «Русский», English → «English (US)».
