«Цель достигнута» — в коде метрика success_at_1: ожидаемое блюдо из банка вопросов входит в число первых упоминаний позиций меню в финальной реплике кассира.

Первая числовая колонка — число диалогов (уникальных вопросов) в строке; далее доля «цель достигнута» в колонках Без RAG и RAG, затем разница в процентных пунктах (RAG − Без RAG). В этой HTML-таблице намеренно только эти столбцы (удобнее для слайда). Доли по флагу hallucination у судьи LLM по режимам и столбец «значимо» (McNemar по цели) — в summary_table.md (тот же прогон make reports / analyze_question_experiments). Для малых выборок отсутствие значимости по McNemar не означает отсутствие эффекта.
