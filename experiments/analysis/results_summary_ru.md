# Итоги и выводы (кратко)

## Условия эксперимента

- Сравниваются два режима подачи меню кассиру: **полный JSON в промпте** и **RAG** (векторный поиск фрагментов меню).
- Для каждого типа вопросов взяты **парные** наблюдения: один и тот же вопрос в обоих режимах (пересечение идентификаторов в выгрузках).
- Всего диалогов в сравнении по всем типам: **1331**.

## Главный результат (по всей выборке)

- **Доля ответов, где цель вопроса достигнута** (ожидаемое блюдо — среди первых упоминаний меню в ответе): **54.55%** (без RAG) → **70.62%** (RAG), разница **+16.07** п.п. 95%-е доверительные интервалы (Уилсон): [51.9–57.2]% и [68.1–73.0]%.
- **Парное сравнение** (McNemar по признаку «цель достигнута»): **статистически значимый** сдвиг в пользу RAG (p практически 0).
- **Доля ответов, которые судья LLM пометил как проблемные** (поле `hallucination` в `metrics`): **14.88%** → **4.73%**, изменение **-10.15** п.п.; McNemar по этому признаку: **p < 0.05**.

## По типам сценариев

- **Аллергены / исключения** (`allergy`, N = 305): цель достигнута 44.26% → 71.8% (+27.54 п.п.); судья LLM пометил 23.93% → 6.56% (-17.37 п.п.); McNemar по «цели»: значимо (p < 0.05).
- **Диетические ограничения** (`diet`, N = 223): цель достигнута 53.81% → 59.64% (+5.83 п.п.); судья LLM пометил 11.66% → 5.38% (-6.28 п.п.); McNemar по «цели»: не значимо при N=223 (p = 0.1983).
- **Заказ на несколько человек** (`group`, N = 159): цель достигнута 52.83% → 62.26% (+9.43 п.п.); судья LLM пометил 11.32% → 13.84% (+2.52 п.п.); McNemar по «цели»: не значимо при N=159 (p = 0.0966).
- **Разговорная речь: исключения и уточнения («без лука», «уберите соус»)** (`lexical`, N = 144): цель достигнута 40.28% → 77.78% (+37.5 п.п.); судья LLM пометил 38.89% → 3.47% (-35.42 п.п.); McNemar по «цели»: значимо (p < 0.05).
- **Смешанный сценарий** (`mixed`, N = 175): цель достигнута 14.29% → 36.0% (+21.71 п.п.); судья LLM пометил 3.43% → 2.29% (-1.14 п.п.); McNemar по «цели»: значимо (p < 0.05).
- **Простой заказ (название блюда)** (`simple`, N = 325): цель достигнута 93.54% → 96.62% (+3.08 п.п.); судья LLM пометил 5.85% → 0.0% (-5.85 п.п.); McNemar по «цели»: не значимо при N=325 (p = 0.089).

## Формулировки для заключения / устного доклада

1. На объединённой выборке парных вопросов режим с **RAG** даёт **более высокую** долю ответов, в которых выполнена цель сценария, и **ниже** долю ответов, которые **судья LLM** пометил как проблемные (`hallucination`), по сравнению с режимом **без RAG** (полное меню в промпте).
2. Усиление эффекта заметнее в отдельных типах сценариев (см. таблицу и графики); там, где парный тест не достиг уровня 5%, осторожно формулировать вывод как **тенденцию**, а не как доказанное отличие на данном N.
3. Оценка качества опирается на **автоматические метрики** в JSON вопросов и **модель-судью**; при сбое разбора ответа судьи используется эвристика (см. `metric_sources` в исходных файлах).
